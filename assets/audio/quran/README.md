# Quran Audio Files

This directory should contain Quran recitation audio files.

## File Structure:
- Each surah should have its own audio file
- File naming convention: `001.mp3`, `002.mp3`, etc. (surah number with leading zeros)
- Files should be in MP3 format for best compatibility

## Example:
- `001.mp3` - Al-Fatihah
- `002.mp3` - Al-Baqarah
- `003.mp3` - Ali 'Imran
- etc.

## Note:
Due to file size limitations, actual audio files are not included in this repository.
You can download Quran recitations from sources like:
- https://www.mp3quran.net/
- https://quran.com/
- Other Islamic audio resources

Make sure to respect copyright and use only permissible sources.
